1. install pygame library by the following command
   
   > pip install pygame

2. open visual studio code and create a new file called main.py

3. If you want to release the exe file, you can install auto-py-to-exe 
  
   > pip install auto-py-to-exe
   
   > auto-py-to-exe
   
4. Download NSIS to build a stand-alone setup file
